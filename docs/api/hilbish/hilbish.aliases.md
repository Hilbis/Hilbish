## Introduction
The alias interface deals with all command aliases in Hilbish.

## Functions
### add(alias, cmd)
This is an alias (ha) for the `hilbish.alias` function.

### delete(name)
Removes an alias.

### list()
Get a table of all aliases.

### resolve(alias)
Tries to resolve an alias to its command.
